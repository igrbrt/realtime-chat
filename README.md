# RealTime Chat

Tecnologias Utilizadas
----------
- Node.js
- Typescript
- Nest.js
- Websockets
- MongoDB
- Docker
- React

Passos para rodar:
----------
```bash
  1. Clonar repositorio
  2. docker-compose up -d
  3. docker compose logs backend -f
  4. docker compose exec backend sh
  5. npm run seed
```
